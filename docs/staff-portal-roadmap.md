## How to use this doc

1. Move a row to **`In progress`** when someone picks it up (optionally add initials or branch name in **Notes**).
2. Set **`Done`** when merged to your main line; add a short pointer (file/API name) in **Notes** if useful.
3. Use **`Deferred`** when deprioritized; say why in **Notes** so it can be revived later.
